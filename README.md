# anki-template-choices

自用选择题 anki 模板

## 功能

- 选项乱序
- 支持单选/多选
- 判断正确答案
- 在Anki for Windows/AnkiDroid上兼容性可能最好，Ankiweb 上可能会有字体缺失
- 夜间模式
